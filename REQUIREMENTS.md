# REQUIREMENTS

**In order for a device to be supported in mikroSDK 2.0**
**following requirement must be met.**

## FLASH MEMORY

* Minimum 8K [`FLASH1`]

## RAM MEMORY

* Minimum 2K [`RAM1`]
