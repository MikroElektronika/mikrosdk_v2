### Improvements

#### mikroSDK

+ Updated support for USB by updating implementation of thirdparty [tinyUSB](https://github.com/hathach/tinyusb) driver
  + Updated files and implementations for all supported MCUs
    + `STM32`
    + `NXP`
    + `TIVA`
    + `GD32`
    + `PIC32`
    + `PIC24`
  + Following classes updated
    + `audio`
    + `bth`
    + `cdc`
    + `dfu`
    + `hid`
    + `midi`
    + `msc`
    + `net`
    + `usbtmc`
    + `vendor`
    + `video`
